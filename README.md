# Transformation Graph - TrGraph

## Usage

```typescript
const trGraph = new TrGraph();
const result = trGraph.objectTransform({
  name: "Test"
}, {
  name: ["utf8-bytes", "bytes-uint256"]
});
assert.equal(result, {
  transformed: { name: 1415934836n },
  linear: [1415934836n]
});

trGraph.transform(11, ["uin16-bytes"]) // Uint8Array(2) [ 0, 16 ]
```
